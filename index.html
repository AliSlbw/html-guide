Referensi Elemen HTML

Panduan lengkap elemen HTML beserta fungsi, atribut, dan contoh penggunaannya.

<div> Division / Wadah Blok Struktur

Elemen container generik yang bersifat block-level. Digunakan untuk mengelompokkan elemen lain agar bisa diberi style CSS atau dimanipulasi dengan JavaScript. Tidak memiliki makna semantik.

Contoh Penggunaan 
<!-- Pembungkus konten -->
<div class="kartu" id="produk-1">
  <h2>Judul Produk</h2>
  <p>Deskripsi produk di sini.</p>
</div>

<!-- Dengan style inline -->
<div style="background:#f0f0f0; padding:20px;">
  Konten dengan background
</div>
AtributTipeKeterangan
idstringIdentifikasi unik elemen, digunakan CSS/JS
classstringNama kelas untuk styling CSS
stylestringCSS langsung di elemen (inline style)
data-*stringData custom, misal data-id="5"
hiddenbooleanMenyembunyikan elemen
💡
Tips: Gunakan <div> untuk keperluan layout. Kalau kontennya punya makna khusus (navigasi, artikel, dll), lebih baik pakai elemen semantik seperti <nav>, <article>.
<span> Span / Wadah Inline Struktur

Mirip dengan <div>, tapi bersifat inline — tidak memulai baris baru. Berguna untuk memberi style pada sebagian teks di dalam paragraf.

Contoh Penggunaan 
<p>
  Harga normal <span style="text-decoration:line-through">Rp50.000</span>
  sekarang <span style="color:red; font-weight:bold">Rp30.000</span>
</p>
<section> Section / Bagian Konten Struktur

Mengelompokkan konten yang tematik dan terkait satu sama lain. Biasanya memiliki heading di dalamnya. Berbeda dengan <div>, <section> punya makna semantik.

Contoh Penggunaan 
<section>
  <h2>Tentang Kami</h2>
  <p>Kami adalah perusahaan teknologi...</p>
</section>

<section>
  <h2>Produk Kami</h2>
  <div class="grid">...</div>
</section>
<p> Paragraph Teks

Digunakan untuk membuat paragraf teks. Browser otomatis memberi jarak atas dan bawah pada setiap paragraf.

Contoh Penggunaan 
<p>Ini adalah paragraf pertama.</p>
<p>Ini paragraf kedua, dengan spasi otomatis dari paragraf pertama.</p>

<!-- Paragraf dengan align -->
<p style="text-align:center">Teks di tengah</p>
<h1> — <h6> Heading / Judul Teks

Elemen judul dengan 6 tingkatan. <h1> paling besar dan penting (biasanya judul utama halaman), <h6> paling kecil. Penting juga untuk SEO.

Contoh Penggunaan 
<h1>Judul Halaman Utama</h1>
<h2>Sub Judul Bagian</h2>
<h3>Sub-sub Judul</h3>
<h4>Tingkat 4</h4>
<h5>Tingkat 5</h5>
<h6>Tingkat 6 (terkecil)</h6>
⚠️
Perhatian SEO: Gunakan hanya satu <h1> per halaman. Jangan melewati urutan (misal dari h1 langsung ke h4).
<a> Anchor / Tautan Teks

Membuat hyperlink ke halaman lain, file, email, nomor telepon, atau lokasi di halaman yang sama.

Contoh Penggunaan 
<!-- Link ke website -->
<a href="https://google.com" target="_blank">Buka Google</a>

<!-- Link ke halaman lokal -->
<a href="tentang.html">Tentang Kami</a>

<!-- Link ke email -->
<a href="mailto:akbar@email.com">Kirim Email</a>

<!-- Link download file -->
<a href="file.pdf" download>Download PDF</a>

<!-- Lompat ke bagian di halaman yang sama -->
<a href="#kontak">Ke Bagian Kontak</a>
AtributTipeKeterangan
hrefURL/stringTujuan link (URL, mailto:, tel:, #id)
targetstring_blank = tab baru, _self = tab sama
downloadboolean/stringMemaksa file diunduh, bukan dibuka
relstringnoopener noreferrer untuk keamanan _blank
<strong> & <em> Bold & Italic Teks

<strong> menandai teks penting (tampil tebal). <em> menandai penekanan emphasis (tampil miring). Keduanya punya makna semantik, beda dengan <b> dan <i> yang hanya visual.

Contoh Penggunaan 
<p>
  <strong>Perhatian:</strong> Jangan lupa simpan data kamu.
</p>

<p>
  Tekan tombol <em>Simpan</em> untuk menyimpan perubahan.
</p>

<!-- Bisa dikombinasi -->
<strong><em>Sangat penting!</em></strong>
<img> Image / Gambar Media

Menampilkan gambar. Ini adalah void element — tidak punya tag penutup. Atribut alt wajib diisi untuk aksesibilitas dan SEO.

Contoh Penggunaan 
<!-- Gambar lokal -->
<img src="foto.jpg" alt="Foto Profil" width="200"/>

<!-- Gambar dari URL -->
<img src="https://picsum.photos/400/300" alt="Gambar Acak"/>

<!-- Lazy loading -->
<img src="besar.jpg" alt="..." loading="lazy"/>

<!-- Gambar responsif -->
<img src="foto.jpg" alt="..." style="width:100%; height:auto"/>
AtributTipeKeterangan
srcURLPath atau URL gambar (wajib)
altstringTeks alternatif jika gambar gagal muat (wajib)
width / heightnumberUkuran gambar dalam pixel
loadingstringlazy = muat saat mau terlihat, eager = langsung
onerrorJSJalankan kode jika gambar gagal dimuat
<video> Video Player Media

Menyematkan pemutar video langsung di halaman. Mendukung format MP4, WebM, dan OGG. Bisa dikontrol via JavaScript.

Contoh Penggunaan 
<!-- Video dasar -->
<video src="video.mp4" controls></video>

<!-- Video dengan opsi lengkap -->
<video
  width="640"
  height="360"
  controls
  autoplay
  muted
  loop
  poster="thumbnail.jpg">
  <source src="video.mp4"  type="video/mp4"/>
  <source src="video.webm" type="video/webm"/>
  Browser kamu tidak mendukung video.
</video>

<!-- Kontrol via JavaScript -->
<video id="myVid" src="video.mp4"></video>
<button onclick="document.getElementById('myVid').play()">Play</button>
<button onclick="document.getElementById('myVid').pause()">Pause</button>
AtributTipeKeterangan
srcURLPath file video
controlsbooleanTampilkan kontrol play/pause/volume
autoplaybooleanPutar otomatis saat halaman dimuat
mutedbooleanMulai tanpa suara (wajib jika pakai autoplay)
loopbooleanUlangi video dari awal setelah selesai
posterURLGambar thumbnail sebelum video diputar
preloadstringnone / metadata / auto
💡
Tips: Gunakan <source> di dalam <video> untuk menyediakan beberapa format. Browser akan memilih yang didukungnya.
<audio> Audio Player Media

Menyematkan pemutar audio. Mendukung MP3, OGG, dan WAV. Cara penggunaannya mirip dengan <video>.

Contoh Penggunaan 
<audio controls>
  <source src="lagu.mp3" type="audio/mpeg"/>
  <source src="lagu.ogg" type="audio/ogg"/>
  Browser tidak mendukung audio.
</audio>

<!-- Background music (autoplay perlu muted) -->
<audio src="bgm.mp3" autoplay loop muted></audio>
<iframe> Inline Frame Media

Menyematkan halaman web lain, video YouTube, peta Google Maps, atau konten eksternal lainnya di dalam halaman kamu.

Contoh Penggunaan 
<!-- Embed YouTube -->
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  allowfullscreen>
</iframe>

<!-- Embed Google Maps -->
<iframe
  src="https://maps.google.com/maps?q=Jakarta&output=embed"
  width="100%"
  height="400"
  style="border:0">
</iframe>
<form> Form / Formulir Form

Wadah untuk elemen input. Mengatur ke mana dan bagaimana data dikirim ke server.

Contoh Penggunaan 
<form action="/kirim" method="POST">
  <label for="nama">Nama:</label>
  <input type="text" id="nama" name="nama"/>
  <button type="submit">Kirim</button>
</form>
AtributTipeKeterangan
actionURLURL tujuan pengiriman data
methodstringGET (URL) atau POST (body request)
enctypestringmultipart/form-data untuk upload file
<input> Input / Kolom Isian Form

Elemen input serbaguna. Perilakunya berubah sesuai atribut type. Void element (tidak ada tag penutup).

Berbagai Tipe Input 
<!-- Teks biasa -->
<input type="text"     placeholder="Masukkan nama"/>
<input type="email"    placeholder="email@contoh.com"/>
<input type="password" placeholder="Kata sandi"/>
<input type="number"   min="0" max="100"/>
<input type="tel"      placeholder="08xx"/>

<!-- Pilihan -->
<input type="checkbox" id="setuju"/>
<input type="radio"    name="gender" value="pria"/>

<!-- Lainnya -->
<input type="date"/>
<input type="color"/>
<input type="range"    min="0" max="100"/>
<input type="file"     accept="image/*"/>
<input type="hidden"   value="data-tersembunyi"/>
AtributTipeKeterangan
typestringJenis input (text, email, password, number, dll)
namestringNama field, dikirim bersama data form
valuestringNilai default input
placeholderstringTeks panduan saat input kosong
requiredbooleanWajib diisi sebelum form dikirim
disabledbooleanMenonaktifkan input
readonlybooleanHanya bisa dibaca, tidak bisa diedit
maxlengthnumberBatas maksimal karakter
<button> Tombol Form

Membuat tombol yang bisa diklik. Lebih fleksibel dari <input type="button"> karena bisa berisi HTML di dalamnya.

Contoh Penggunaan 
<!-- Submit form -->
<button type="submit">Kirim</button>

<!-- Tombol biasa dengan event -->
<button type="button" onclick="alert('Halo!')">Klik Saya</button>

<!-- Tombol dengan icon (HTML di dalamnya) -->
<button type="button">
  🔍 Cari
</button>

<!-- Tombol nonaktif -->
<button disabled>Tidak Bisa Diklik</button>
<select> Dropdown / Pilihan Form

Membuat menu dropdown pilihan. Setiap pilihan menggunakan tag <option>. Bisa dikelompokkan dengan <optgroup>.

Contoh Penggunaan 
<select name="kota">
  <option value="">-- Pilih Kota --</option>
  <optgroup label="Jawa">
    <option value="jkt">Jakarta</option>
    <option value="sby" selected>Surabaya</option>
  </optgroup>
  <optgroup label="Sumatera">
    <option value="mdn">Medan</option>
  </optgroup>
</select>

<!-- Pilihan multiple -->
<select name="hobi[]" multiple>
  <option value="coding">Coding</option>
  <option value="gaming">Gaming</option>
</select>
<textarea> Area Teks Panjang Form

Input teks multi-baris. Cocok untuk komentar, pesan, atau deskripsi panjang. Ukuran bisa diatur pengguna (resize).

Contoh Penggunaan 
<textarea
  name="pesan"
  rows="5"
  cols="40"
  placeholder="Tulis pesan kamu di sini..."
  maxlength="500"
  style="resize:vertical">
</textarea>
<ul> <ol> <li> Daftar / List List

<ul> = unordered list (bullet), <ol> = ordered list (nomor), <li> = setiap item dalam list.

Contoh Penggunaan 
<!-- Unordered (bullet) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Ordered (nomor) -->
<ol start="1">
  <li>Buka browser</li>
  <li>Ketik URL</li>
  <li>Enter</li>
</ol>

<!-- List bersarang -->
<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>Backend</li>
</ul>
<table> Tabel Data Table

Menampilkan data tabular. Terdiri dari baris (<tr>), sel header (<th>), dan sel data (<td>). Gunakan hanya untuk data, bukan untuk layout halaman.

Contoh Penggunaan 
<table border="1">
  <thead>
    <tr>
      <th>Nama</th>
      <th>Nilai</th>
      <th>Keterangan</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Akbar</td>
      <td>95</td>
      <td>Sangat Baik</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">Total: 1 siswa</td>
    </tr>
  </tfoot>
</table>
Atribut td/thTipeKeterangan
colspannumberGabung beberapa kolom menjadi satu sel
rowspannumberGabung beberapa baris menjadi satu sel
scopestringUntuk aksesibilitas: col atau row
Elemen Semantik HTML5 Structural Semantic

Elemen-elemen ini memberi makna pada struktur halaman — bukan hanya visual. Membantu SEO, aksesibilitas (screen reader), dan keterbacaan kode.

Struktur Halaman Semantik 
<header>
  <nav>
    <a href="/">Beranda</a>
    <a href="/tentang">Tentang</a>
  </nav>
</header>

<main>
  <article>
    <h1>Judul Artikel</h1>
    <time datetime="2025-03-07">7 Maret 2025</time>
    <p>Isi artikel...</p>
  </article>

  <aside>
    <h2>Artikel Terkait</h2>
  </aside>
</main>

<footer>
  <p>&copy; 2025 Website Saya</p>
</footer>
ElemenFungsi
<header>Header halaman atau section (logo, nav utama)
<nav>Blok navigasi / menu
<main>Konten utama halaman (satu per halaman)
<article>Konten mandiri: artikel, posting, komentar
<aside>Konten samping / sidebar
<footer>Footer halaman atau section
<time>Tanggal/waktu dengan format mesin di datetime
<figure>Pembungkus media (gambar, diagram) dengan keterangan
<figcaption>Keterangan/caption untuk <figure>
✅
Best Practice: Gunakan elemen semantik sebisa mungkin. Jangan pakai <div> untuk semua hal — gunakan <nav> untuk navigasi, <article> untuk artikel, <footer> untuk footer.
